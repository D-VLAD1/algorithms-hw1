# algorithms-hw1

## Variant
`V4` (get largest group, change group by email, get more successful group)\
`S3` (sort by name and surname)

## Requirements
Download `CMake 3.27` (and higher) and `C++20` (and higher) to launch tests.

## Usage
```shell
mkdir build
cd build
cmake ..
cmake --build .
```
